# 2、lib包说明
- log4j.jar 代码流程中的日志输出管理
- commons-codec 用来做BASE64编码，消息解密中使用
- json.jar 一种消息格式，用来处理返回的消息
- bcpro-jdk.jar 一个很偏僻的类，用来配合消息解密
